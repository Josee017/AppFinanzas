Finance Flow

Una aplicación de gestión de finanzas personales diseñada bajo el modelo "Offline-First". Prioriza la operación local sin internet, sincronizando datos automáticamente con la nube cuando la conexión se restablece.

Manual de Usuario

1. Primeros Pasos

Al iniciar, lo primero es configurar tu entorno financiero. La aplicación necesita saber dónde tienes el dinero y en qué lo gastas.

  • Ve a la pestaña Billeteras y crea cuentas (ej. "Efectivo", "Banco").
  • Ve a Categorías y define tus gastos (ej. "Comida", "Transporte").
[UI: Listado de Billeteras]

2. Registrando Transacciones

Para registrar un movimiento financiero:

  1. Pulsa el botón flotante "+".
  2. Selecciona Ingreso o Gasto.
  3. Elige la Billetera de origen/destino.
  4. Elige la Categoría (se filtran según el tipo).
  5. Ingresa monto, fecha y nota.
  6. Pulsa Guardar.

Nota: Si estás offline, verás un icono de sincronización pendiente.

Solución de Errores Comunes

Error DB9: ignoreDuplicate not allowed

Causa: RxDB v14+ no permite ignoreDuplicate: true en producción.

Solución: Eliminar esa propiedad en src/lib/rxdb.ts. El patrón Singleton ya maneja esto.

Netlify 404 en rutas (Refresh)

Causa: SPA Routing no configurado en servidor estático.

Solución: Crear netlify.toml con redirección /* -> /index.html.

Usuario "atrapado" en Registro

Causa: Condición de carrera entre AuthContext y React Router.

Solución: Usar window.location.href en lugar de navigate() tras el registro para forzar una recarga limpia.
